🚀 从零搭建你的 GitHub 博客

本教程将手把手教你如何创建一个属于自己的 GitHub 个人博客网站。

步骤一：准备工作

  • 注册一个 GitHub 账号
  • 安装 Git（建议）

步骤二：创建仓库

  1. 登录 GitHub，点击右上角 New repository
  2. 命名仓库为：yourusername.github.io
  3. 点击 Create repository 创建

步骤三：上传网站文件

方法一：用 Git 上传

git clone https://github.com/yourusername/yourusername.github.io
# 将 index.html 等网页文件拷进去
git add .
git commit -m "首次上传"
git push

方法二：在 GitHub 网站上上传

  1. 进入仓库页面，点击 Upload files
  2. 上传网页文件，提交

步骤四：开启 GitHub Pages

  1. 点击仓库 > Settings > Pages
  2. 选择部署分支为 main，文件夹选 root
  3. 几分钟后即可访问 https://yourusername.github.io

🎉 恭喜你，猪猪也成功搭建好了自己的博客啦！
